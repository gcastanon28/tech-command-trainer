# **App Name**: CodePath

## Core Features:

- Command & Concept Repository: A searchable collection of Linux and Python commands and concepts with clear explanations and usage examples.
- Interactive Flashcards: Review command syntax, descriptions, and examples through a flashcard system, allowing users to mark items as mastered.
- Customizable Quizzes: Generate multiple-choice or fill-in-the-blank quizzes based on user-selected topics (Linux, Python, difficulty levels) to test knowledge.
- AI-Generated Daily Challenge: Receive a unique daily coding or command-line task, personalized by a generative AI tool to reinforce diverse learning areas.
- Progress Dashboard: Visualize learning progress, completed quizzes, flashcard mastery statistics, and daily challenge streaks to track user improvement.
- User Authentication: Allow users to securely create accounts and log in to persist their learning progress and preferences across sessions.

## Style Guidelines:

- Primary color: A vibrant, tech-inspired blue (#4796F0) to evoke focus and digital expertise against a dark theme.
- Background color: A subtly dark, cool grey (#191F25) to provide a clean and calm backdrop, enhancing content readability.
- Accent color: A bright aqua (#68E1FF) to highlight interactive elements, calls to action, and important information, ensuring strong contrast.
- Headline font: 'Space Grotesk' (sans-serif) for a modern, techy aesthetic. Body font: 'Inter' (sans-serif) for excellent legibility and a clean feel in content and explanations. Code snippets: 'Source Code Free' (monospace) for clear representation of commands.
- Utilize minimalist, outlined icons that align with the 'clean dark UI' aesthetic, ensuring they are intuitive and easily recognizable for navigation and content representation.
- Implement a clean, content-focused layout with a clear hierarchy. A prominent sidebar for navigation and a primary content area to ensure command and quiz details are easily digestible. Ensure responsive design for various screen sizes.
- Incorporate subtle, non-distracting transitions for interactive elements like flashcard flips, quiz feedback, and progress updates, enhancing user experience without overwhelming the learner.
